## 遊びを本気で作り、本気で届ける。

T-Projectは、Minecraftを中心としたコンテンツを企画・開発・運営するチームです。
配信者向けサーバーの構築からプラグイン開発、イベント設計まで、「面白い」を形にするための一連の工程を自分たちの手で担っています。

## Values

- **思いついたらまず作る** — 議論より先にプロトタイプを出す。
- **一人の手柄よりチームの作品** — 個人の名前より、出来上がったものの質で語る。
- **届かなければ作ってないのと同じ** — 完成しただけでは意味がない。届けるところまでが仕事。

## What We Do

- Minecraftサーバーの企画・構築・運用
- Spigot / Paper プラグインの設計・開発
- 配信連動システム（妨害マイクラ等）のライセンス提供
- Web アプリ・ツールの開発

## Tech Stack

Kotlin / Java (Spigot/Paper) / TypeScript / Next.js / React / PostgreSQL / MariaDB / Docker / GitHub Actions
